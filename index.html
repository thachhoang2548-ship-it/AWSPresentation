ARCHITECTURE
OVERVIEW

Kiến trúc hệ thống gồm 3 layer chính được bóc tách độc lập để đảm bảo hiệu năng:

1. Frontend: CDN + Hosting

2. Backend: API + Compute

3. Database & Services

Nhờ tích hợp loạt dịch vụ S3, EC2, RDS, Cognito, hệ thống đạt độ tin cậy tuyệt đối.

FRONTEND FLOW

Luồng Frontend tĩnh đảm bảo tải mượt mà dưới 1 giây. User request đi vào hệ thống sẽ qua các trạm:

01.

Amazon CloudFront

Đóng vai trò CDN (Content Delivery). Tối ưu Latency toàn cầu.

02.

Amazon S3

Host tĩnh Frontend ReactJS với độ bền dữ liệu (Durability) 99.999%.

HIGH AVAILABILITY
& SCALING

Hệ thống "Không bao giờ chết" nhờ các cơ chế đặc thù:

Auto Scaling

Scale EC2 theo % lượng traffic, luôn sẵn sàng mùa cao điểm.

Multi-AZ Database

Amazon RDS đứng ở 2 Data Center khác nhau (Primary + Standby). Tự động Failover nếu cháy nổ máy chủ.

SECURITY LAYERS

01.

Security Group

Wall rào 80/443. RDS cấm toàn bộ Internet tải về, chỉ cho phép EC2.

02.

AWS WAF

Lá chắn Web chặn Hacker dùng SQL Injection & XSS.

03.

Private Subnet

Cơ sở dữ liệu Database nằm trong bóng tối mạng ngầm (Không mở Public).

END-TO-END SUMMARY

1 User -> CloudFront -> S3 (Tải Frontend)
2 Frontend gọi lên API (Route53 -> ALB)

5 Server phản hồi nhanh chóng lên màn hình User!
